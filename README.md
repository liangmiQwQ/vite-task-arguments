Run the commands and compare the output:

```bash
vp run dev say 1
```

```bash
vp run dev -b say 1
```

```bash
vp run dev --help
```

---

`nr` can be replaced with `pnpm run`, `yarn run`, `bun run`

```bash
nr dev say 1
```

```bash
nr dev -b say 1
```

```bash
nr dev --help
```

---

```bash
npm run dev say 1
```

```bash
npm run dev -b say 1
```

```bash
npm run dev -- -b say 1
```

```bash
npm run dev -- --help
```
